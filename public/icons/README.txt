Icônes PWA et app mobile
========================

Pour que l’app soit installable (PWA) et pour les stores, ajoutez ici des PNG aux tailles suivantes :

  icon-72.png   (72x72)
  icon-96.png   (96x96)
  icon-128.png  (128x128)
  icon-144.png  (144x144)
  icon-152.png  (152x152)  — Apple touch
  icon-192.png  (192x192)  — Android / Apple
  icon-384.png  (384x384)
  icon-512.png  (512x512)  — splash / stores

Vous pouvez générer toutes les tailles à partir d’une image 512x512 avec :
  https://www.pwabuilder.com/imageGenerator
  ou https://realfavicongenerator.net/

Utilisez le même logo / visuel pour toutes les icônes.
