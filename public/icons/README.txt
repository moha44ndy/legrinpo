Icônes PWA et app mobile
========================

Les PNG 192x192 et 512x512 sont générés à partir de favicon.svg avec :
  node scripts/generate-pwa-icons.js

Fichiers utilisés par le manifest PWA :
  icon-192.png  (192x192) — Android / Apple
  icon-512.png  (512x512) — splash / stores

Pour d'autres tailles (72, 96, 128, 144, 152, 384), vous pouvez utiliser :
  https://www.pwabuilder.com/imageGenerator
  ou https://realfavicongenerator.net/

Utilisez le même logo / visuel pour toutes les icônes.
